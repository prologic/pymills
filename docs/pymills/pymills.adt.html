class Queue
    Queue Class - FIFO (First In First Out)
 
This class is simply an extenstion or encapsulation of a list.
It provides the basic functionality of a standard Queue.
 
Example:
>>> import adt
>>> queue = adt.Queue()
>>> queue.push("foo")
>>> queue.push("bar")
>>> queue.peek()
'foo'
>>> queue.pop()
'foo'
>>> queue.empty()
False
>>> queue.pop()
'bar'
>>> queue.empty()
True
