Timers
 
This is a module that allows you to implement timed-events
in your running application/process.
 
Example:
from pymills.timers import Timers
 
def hello(name, length, args):
   print "Hello World"
 
timers = Timers()
timers.add("hello", 5, hello)
 
while True:
   timers.run()
